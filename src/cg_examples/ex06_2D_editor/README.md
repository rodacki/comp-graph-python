# 2D Geometric Shape Editor

This is a small academic project, a 2D geometric shape editor, developed using OpenGL and GLUT with the pyopengl library in Python.

## Table of Contents
- [Description](#description)
- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)
- [License](#license)

## Description
The 2D Geometric Shape Editor is a Python application that allows users to create and manipulate simple geometric shapes using OpenGL and GLUT. It provides a user-friendly interface for creating shapes such as squares, circles, triangles and non-convex polygons, and offers features like resizing, rotation, and color changes.

## Features
- Create and draw various geometric shapes individually or in combination.
- Manipulate shapes by resizing, rotating, and changing colors.
- Simple user interface for easy shape creation and manipulation.
- Real-time rendering using OpenGL for fast and interactive editing.

## Installation
To run the 2D Geometric Shape Editor, make sure you have the following prerequisites installed:

1. Python (version 3.6 or higher)
2. OpenGL library
3. GLUT library
4. PyOpenGL package
5. NumPy package

After installing the prerequisites, follow these steps:

1. Clone or download this repository.
2. Navigate to the project directory.
3. Run the main.py file using the Python interpreter.

## Usage
1. Launch the editor by running the main.py file.
2. Use the user interface to select the desired shape.
3. Click on the canvas to create a shape at the specified position.
4. Use the provided tools to manipulate the shape (resize, rotate, change color, etc.).
